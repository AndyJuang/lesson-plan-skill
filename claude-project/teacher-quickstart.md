# 老師使用速查表（A4 一頁）

> **發給老師用**：這份可以直接列印 A4 放在辦公桌上，或貼進 LINE 群組作為備忘。

---

## 🎯 我可以請教案生成助手做什麼？

一次性產出一份完整教案，包含：
**教師手冊｜10 頁簡報大綱｜學習單｜測驗題｜圖像提示詞｜雙語對話（語言科）**

---

## 📝 使用 3 步驟

### Step 1. 進入 Project

到 [claude.ai](https://claude.ai) → 左側「Projects」→ 點「{{你的 Project 名稱}}」。

### Step 2. 貼上任務模板

```
我要做一份教材。

【來源資料】
（選一：貼上單字表／附加課本 PDF／描述單元章節內容）

【授課需求】
- 學科：英語 / 自然 / 社會 / 本土語（請擇一）
- 單元：{{主題名稱}}
- 年級：{{幾年級}} / CEFR {{程度}}
- 時長：{{分鐘數}}
- 文化錨點：{{如：泰雅雲豹傳說；若無請留空}}
- 特殊要求：{{如：雙語對照、強化口說；若無請留空}}
```

### Step 3. 等待產出 + 複製

Claude 會分段產出 6–7 份內容，每段以 `# 📘 01_教師手冊` 等標題開頭。

複製整段 → 貼到 Google 文件／Word → 完成！

## 💡 實用範例指令

### 範例 A：英語 × 泰雅文化

```
我要做一份教材。

【來源資料】
動物單字表：cloud leopard, black bear, sambar deer, forest, mountain

【授課需求】
- 學科：英語
- 單元：Animals
- 年級：三年級 / CEFR Pre-A1
- 時長：40 分鐘
- 文化錨點：泰雅雲豹傳說
- 特殊要求：雙語對照、族語問候
```

### 範例 B：自然 × 在地生態

```
我要做一份教材。

【來源資料】
四年級自然第 3 單元「校園植物觀察」章節（我會附上課本影本）

【授課需求】
- 學科：自然
- 單元：校園植物觀察
- 年級：四年級
- 時長：2 節連堂（80 分鐘）
- 文化錨點：東澳常見原生植物（山棕、姑婆芋）
- 特殊要求：需有戶外觀察學習單
```

### 範例 C：本土語 × 節慶

## ❓ 常見問題

### Q1：Claude 說它沒看到範本怎麼辦？
A：通知 Project 擁有者檢查「Project knowledge」是否已上傳 6 份範本。

### Q2：產出的族語拼寫正確嗎？
A：**一定不要直接當正確答案**。所有族語拼寫都會標 `★`，請務必請族語老師確認。

### Q3：文化典故是真的還是 AI 編的？
A：AI 產出都是初稿，文化典故可能是 AI 參考一般資料推論而來。**必須請部落耆老或文健站複核**後才能正式上課。

### Q4：簡報可以直接用嗎？
A：產出是**簡報大綱**（Markdown 格式）。請把整份貼進 **Gemini Canvas** 或 **Google Slides + Gemini 側邊欄**，讓它轉成實際投影片。

### Q5：圖像提示詞是什麼？要做什麼？
A：是一組英文指令，複製後貼到 **Nanobanana** 或 **Gemini Image** 就會生成對應的單字圖卡或故事插圖。Claude 不會直接產圖，但會寫好提示詞給你用。

### Q6：可以改產出的內容嗎？
A：可以。產出後直接在對話接著說「請把第 3 個單字改成 eagle」或「學習單第 4 題改難一點」，Claude 會依你的指示調整。

### Q7：我想新增一個範本（例如家長聯絡單）怎麼辦？
A：請通知 Project 擁有者，把新範本加進 `claude-project/` 資料夾並上傳到 Project knowledge。

## ⚠️ 絕對要遵守的兩件事

1. **族語／文化內容上課前要複核**：AI 會標 `★`，請找族語老師或耆老確認。
2. **敏感學生資訊不要貼進來**：學生姓名、成績、家庭背景不要輸入對話框，用代號即可（如「學生 A」）。
