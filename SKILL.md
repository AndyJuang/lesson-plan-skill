# 教案生成 (Lesson Plan Package Generator)

依使用者提供的「來源資料」與「授課需求」，一次性產出完整教案包，包含教師手冊、簡報大綱、學習單、測驗題目、圖像提示詞、（如為語言科）雙語對話腳本。

## 觸發時機

當使用者提供以下兩組輸入時啟用本 Skill：

1. **來源資料（Source）**：課本 PDF、單元章節文字、單字表、參考網址、現有教材、NotebookLM 抽出之摘要⋯⋯任何可作為教學內容依據的素材。
2. **授課需求（Requirements）**：
   - 學科與單元主題（如：英語 / Animals）
   - 學生年級與程度（如：三年級 / CEFR Pre-A1）
   - 節數與時長（如：1 節 40 分鐘）
   - 文化錨點或融入主題（如：泰雅族雲豹傳說；可留空）
   - 特殊要求（如：雙語對照、強化口說、融入部落生活場景）

使用者說「幫我做教案」「生成教材包」「把這個單元做成完整教案」「lesson plan」等皆可啟動。

## 輸出成果

於 `./教案輸出/<主題>_<YYYYMMDD>/` 產出 7 份檔案：

| 檔名 | 內容 | 備註 |
|---|---|---|
| `01_教師手冊.md` | 教學流程、時間分配、學習目標、文化把關提醒 | 必產 |
| `02_簡報大綱.md` | 10–15 頁投影片，每頁含標題、內文、圖像位置 | 必產 |
| `03_學習單.md` | 5–8 題學生練習，含配對、填空、閱讀、文化思考 | 必產 |
| `04_測驗題目.md` | 10–15 題形成性評量，含單選、短答、應用 | 必產 |
| `05_圖像提示詞.md` | 10–15 組 Nanobanana／Gemini 圖像提示詞（英文） | 必產 |
| `06_雙語對話腳本.md` | 師生／學生之間的情境對話示例 | 僅語言科產出 |
| `INDEX.md` | 檔案清單、使用流程、使用前檢核 | 必產 |

## 工作流程

依 `workflow.md` 執行 5 階段：

1. **Intake（確認輸入）**：盤點來源資料與授課需求，列出缺漏並詢問。
2. **Extract（抽取骨架）**：從來源資料萃取核心單字、句型、知識點、學習目標。
3. **Generate（生成檔案）**：依 `templates/` 範本產出 6–7 份檔案，可平行化。
4. **Lint（繁中潤稿）**：每份中文檔案執行 `mcp__zhtw-mcp__zhtw`（`lexical_contextual`）。
5. **Package（打包交付）**：寫 `INDEX.md`、回報資料夾路徑與摘要。

## 規則

- **語言**：全部繁體中文、台灣慣用語；每份中文內容必跑 `mcp__zhtw-mcp__zhtw` 潤稿。
- **圖像提示詞**：一律英文（Nanobanana／Gemini Image 相容），每組開頭標註 `[單字卡]`／`[故事場景]`／`[封面]`／`[活動圖]`。
- **文化敏感度**：若使用者提供文化錨點（如泰雅、阿美、排灣等原住民族／特定地區文化），必須：
  - 在 `01_教師手冊.md` 加上「三層審核提醒」（AI 初稿 → 老師改寫 → 在地耆老／族語老師／文健站複核）。
  - 文化典故若為 AI 推論，檔案中以 `★ 待在地確認` 標記。
  - 族語拼寫一律標註 `★ 拼寫待族語老師確認`。
- **年齡適切性**：依年級自動調整詞彙量、句長、概念抽象度。
- **無捏造事實**：歷史／文化／科學事實若非來源資料明確載明，標為「AI 推論，建議查證」。
- **資料夾路徑**：預設 `./教案輸出/<主題>_<YYYYMMDD>/`；使用者可指定其他路徑。
- **TaskCreate**：每階段建立一個任務，`in_progress` 開始、`completed` 結束。

## 預設值（可覆寫）

- 簡報頁數：10 頁（1 封面 + 5 內容 + 2 對話／應用 + 1 文化背景 + 1 活動／總結）
- 測驗題數：10 題（單選 6 + 短答 3 + 應用 1）
- 學習單題型：配對、填空、造句、閱讀理解、文化思考各一至兩題
- 教學時長：40 分鐘（1 節）
- 年級預設：國小中年級（三至四年級）

## 回報格式

每階段完成後以短報告回覆：

```
✅ Phase {N}/{5} 完成：{階段名稱}
產出：{檔名或動作}
下一步：{next action}
```

全部完成後給完整清單：

```
🎯 教案包已生成於：{folder path}

📂 檔案清單：
  01_教師手冊.md       ({行數} 行)
  02_簡報大綱.md       ({頁數} 頁)
  ...

⚠️ 使用前檢核：
  - 族語拼寫 × {N} 處待確認
  - 文化典故 × {M} 處待在地複核
  - 推薦審核對象：{e.g., 族語老師、文健站}
```
