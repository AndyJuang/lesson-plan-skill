# 教案生成 Skill

> Claude Code Skill — 餵一份課本或單元素材與授課需求，**一次性**產出完整教案包：教師手冊、簡報大綱、學習單、測驗題目、圖像提示詞、雙語對話腳本。

適用對象：國小／國中教師、師資培育講師、校本特色課程設計者。
開發初衷：把「備課的重複性打字與排版工作」交給 AI，讓老師回到設計學習經驗與理解孩子的本業。

---

## 🎯 這個 Skill 解決什麼問題？

傳統備課流程：老師花 2–3 小時翻課本 → 抄單字 → 找圖 → 打學習單 → 排簡報 → 出測驗。
有了本 Skill：**一個指令、3–5 分鐘**，產出一份可帶回教室立即使用的完整教材包。

### 支援情境

- ✅ 國小／國中英語課（含雙語對話腳本）
- ✅ 本土語言、族語融合課程（含文化敏感度內建機制）
- ✅ 校本特色課程（提供文化錨點即可）
- ✅ 其他學科（自然、社會、藝文——對話腳本檔案會自動略過）

## 📂 Skill 結構

```
教案生成/
├── SKILL.md              ← 觸發條件、輸出規格、規則
├── workflow.md           ← 5 階段協議：Intake → Extract → Generate → Lint → Package
└── templates/
    ├── 01_teacher_notes.md    ← 教師手冊（流程／目標／文化把關提醒）
    ├── 02_slides_outline.md   ← 10 頁簡報大綱（可再丟給其他簡報 Skill 產 PPTX）
    ├── 03_worksheet.md        ← 學習單（5 大題型）
    ├── 04_quiz.md             ← 測驗題目（單選 6 + 短答 3 + 應用 1）
    ├── 05_image_prompts.md    ← 12 組英文圖像提示詞（Nanobanana／Gemini Image 相容）
    └── 06_dialogue_script.md  ← 3 段雙語對話腳本（淺→中→深）
```

## 🚀 安裝

### 方法一：Claude Code Skill（終端機使用者）

```bash
cd ~/.claude/skills/
git clone https://github.com/AndyJuang/lesson-plan-skill.git 教案生成
```

### 方法二：下載 zip 解壓

1. 在 GitHub 頁面點 `Code` → `Download ZIP`。
2. 解壓後把整個資料夾改名為 `教案生成`，放到 `~/.claude/skills/` 下。

### 方法三：Claude Project 版（零終端機、適合多人共用）

若你的團隊只會用瀏覽器、不想裝 CLI，改走 Claude Project 路線：

1. 進入 [`claude-project/`](claude-project/) 資料夾。
2. 依 [claude-project/README.md](claude-project/README.md) 的 5 分鐘流程設定。
3. 邀請同學年老師加入你的 Project，所有人共用同一套 Skill 邏輯。

### 驗證安裝

開啟 Claude Code，輸入 `/skill`，應該能在清單中看到：

```
教案生成: 教案生成 (Lesson Plan Package Generator)
```

## 🧑‍🏫 使用方式

### 基本指令

```
/skill 教案生成
來源：Animals_Unit.pdf
需求：三年級、CEFR Pre-A1、40 分鐘、融入泰雅雲豹傳說
```

或者用自然語言：

```
請幫我用教案生成 Skill 做一份教材。
我帶了三年級英語第四單元 Animals 的課本 PDF（animals.pdf），
希望融入泰雅族雲豹傳說，學生程度是 CEFR Pre-A1，
一節課 40 分鐘。
```

### 必備輸入

| 項目 | 說明 | 範例 |
|---|---|---|
| 來源資料 | PDF、單字表、章節內容皆可 | `animals.pdf` |
| 學科與單元 | 學科與單元主題 | 英語 / Animals |
| 年級與程度 | 年級 + CEFR 級別（或學科程度） | 三年級 / CEFR Pre-A1 |
| 節數與時長 | 預計上幾節課、每節多久 | 1 節 40 分鐘 |

### 可選輸入

| 項目 | 說明 | 範例 |
|---|---|---|
| 文化錨點 | 融入的在地／族群文化元素 | 泰雅族雲豹傳說 |
| 特殊要求 | 其他教學考量 | 雙語對照、強化口說、融入部落生活場景 |

## 📦 輸出內容

執行完成後會產生資料夾 `./教案輸出/<主題>_<YYYYMMDD>/`，內含 7 份檔案：

| 檔名 | 內容 | 用途 |
|---|---|---|
| `INDEX.md` | 檔案清單與使用前檢核 | 老師先看這一份 |
| `01_教師手冊.md` | 教學流程、時間分配、文化把關提醒 | 上課前讀一遍 |
| `02_簡報大綱.md` | 10 頁投影片逐頁內容 | 貼進 Gemini Canvas / Google Slides |
| `03_學習單.md` | 5 題型練習卷 | 列印給學生 |
| `04_測驗題目.md` | 10 題形成性評量 | 下一節課用 |
| `05_圖像提示詞.md` | 12 組英文圖像提示詞 | 貼進 Nanobanana / Gemini Image |
| `06_雙語對話腳本.md` | 3 段師生對話 | 語言科才產出 |

## 🎨 設計特色

### 1. 文化敏感度內建

若帶入原住民族或特定地區文化錨點，Skill 會：

- 在教師手冊加入**三層審核提醒**（AI 初稿 → 老師改寫 → 族語老師／耆老／文健站複核）
- 族語拼寫一律標 `★` 留待在地老師填入
- 圖像提示詞內建「避免人物正面特寫、避免族群紋樣混用」指引
- 文化典故若為 AI 推論，明確標為「待在地確認」

### 2. 繁體中文 + 台灣教育用語

所有中文輸出自動經 `mcp__zhtw-mcp__zhtw` 潤稿（若環境有此 MCP），並以 `lexical_contextual` 模式確保教育領域用語正確。

### 3. 可重複使用的變數填空範本

每份範本都是 `{{變數}}` 填空格式，Skill 會依輸入自動替換。老師也可以離線複製範本，手動填寫作為 AI 備案。

### 4. 5 階段協議，每階段回報

- **Phase 1 Intake**：盤點輸入、列出缺漏
- **Phase 2 Extract**：從來源抽出單元骨架
- **Phase 3 Generate**：平行產出 7 份檔案
- **Phase 4 Lint**：繁中潤稿
- **Phase 5 Package**：寫 INDEX、回報完整路徑

## 🔗 搭配工具建議

| 工具 | 搭配方式 |
|---|---|
| **NotebookLM** | 先把課本 PDF 上傳為知識庫，再把抽出的摘要當作 Skill 的來源資料 |

## 📸 示範情境

### 情境 A：英語 × 泰雅文化

```
來源：三年級英語 Unit 4 Animals 課本 PDF
需求：40 分鐘、Pre-A1、融入泰雅雲豹傳說、雙語對照
```

產出：以「Cloud Leopard」為主角的單字卡、部落山林故事、族語 ★ 標記、三層審核提醒。

### 情境 B：自然 × 校本課程

```
來源：四年級自然「校園生態」章節
需求：一節課 40 分鐘、加入東澳校園植物觀察
```

產出：單字／名詞對照、學習單、測驗（無對話腳本，因非語言科）。

### 情境 C：本土語 × 節慶

```
來源：五年級本土語「過年」單元
需求：2 節連堂、強化家族稱謂、融入泰雅 Gaga 倫理
```

產出：雙語對話、稱謂配對學習單、文化思考測驗題。

## ⚠️ 注意事項

- **AI 產出都是初稿**：任何文化、歷史、族語內容請務必請在地顧問複核。
- **圖像提示詞為建議值**：實際生成圖像品質依工具而異，建議多 reroll 幾次。
- **Skill 不會自動生成 PPTX 或 PNG**：產出為 Markdown 檔案，提供給下游工具使用。
- **對話腳本只有語言科產出**：其他學科會自動略過這一份。

## 🛠 進階：修改範本

若你的學校有特定格式需求（如固定的學習單標頭、校徽位置），可直接修改 `templates/` 下的 `.md` 檔。Skill 會在下次執行時採用新範本。

常見客製化：
- 在 `01_teacher_notes.md` 加入校本課程評量指標
- 在 `03_worksheet.md` 改為雙面 A4 版型
- 在 `05_image_prompts.md` 調整共通風格宣言（如改成插畫風、剪紙風）
